# Test Case Generator AI Repository

This repository hosts the AI Test Case Generator projects, organized into targeted environments and implementations.

## 📂 Repository Structure

The current folder structure is divided into two primary subdirectories based on different development focuses or agents:

- **`Antigravity/`**: Contains the implementations and projects handled by or related to the Antigravity agent model.
- **`VisualStudioCode/`**: Contains the Visual Studio Code-focused implementations of the Local LLM Test Case Generator projects, including frontend architectures, backend APIs, and integrations.

## 🚀 Projects Included

- **Project01_LocalLLM_TC_Generator** (Located in both folders)
- **Project02_LocalLLM_TC_Gemerator** (Inside VisualStudioCode folder)

For deep instructions on the setup, running instructions, and architecture of a specific sub-project, navigate to the respective directory and read its local `README.md`.
